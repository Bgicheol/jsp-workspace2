1. Eclipse Enterprise Edition

  • for Web Developers.. 붙은 버전으로 다운로드
  • File - New - Dynamic Web Project를 눌러 새 프로젝트를 생성

2. Apache Tomcat Web Application Server의 절차

  • Web Application Server (WAS)
  • 9.0으로 설치
  • 압축 해제 후 새 Dianmic Web Project 생성시 경로를 지정해줘야 한다

3. 웹 애플리케이션 서버 콘솔에서 실행해보기

  • cmd -> cd tomcat/bin -> startup.bat 실행
  • 환경변수가 설정되어 있지 않은 경우 내 pc(내컴퓨터) 우클릭 -> 속성 -> 고급 설정 -> 환경변수에서 JAVA_HOME을 환경 변수로 등록 (값은 jdk경로)
  • 포트번호가 겹치는 경우 tomcat/conf/server.xml에서 포트번호 8080을 다른 변수로 수정(ex:8090)
  • tomcat/webapps 에 원하는 파일을 등록하려면 서버에 접속하는 사용자들에게 해당 페이지를 서비스할 수 있다

4. 새 프로젝트 생성하기

  • File - New - Dynamic Wep Project
  • 프로제트 이름 설정
  • 어떤 WAS를 사용할지 결정
  • next 두번 눌러서 generate web.xml 체크 후 프로젝트 생성
  • web.xml 체크하지 못하고 만든 경우 : 프로젝트 이름 우클릭 - Java EE Tools

5. 새 프로젝트에 접속하기

  • Servers - 우리가 등록한 서버 우클릭 - Add and Remove - 실행할 ㅡ로젝트 abb
  • 서버 우클릭 후 start
  • 웹 브라우저에 직접 주소를 입력해 접ㅂ속할 수 있다 (http://localhost:8090/프로젝트명/webapp내부경로)
  • localhost를 IP주소로 바꿔서 접속할 수 있다 (IP주소 알아내는 방법 - cmd - ipconfig/all)
  • localhost 대신 127.0.0.1을 사용할 수있다
  • WEB-INF 폴더 밑에 페이지를 생성하면 접속이 불가능하다
